德國水道 German channel

    德國水道全長約360公尺喔，是二次世界大戰德國佔領帛琉時所炸出來的水道，目的是要疏通 Koror、Peleliu、Angaur 之間的交通

    帛琉群島的四周是由廣大的淺礁區 (稱做環礁) 環繞, 因此漲退潮的海水會經由很多條天然形成的渠道進出 (稱做Channel)
    這連接外海與內海的天然水道,在生態上扮演了一個重要的角色.
    當漲潮時,外海的海水須經過各地的水道在6小時內完成注滿內海,退潮時內海的海水也會在6小時內流到外海,因此水道的海流經常是非常強勁的.

    德國水道位於帛琉西南方,1900年,德國一所礦業公司為了到南端的小島開採磷礦,在這天然的淺礁區挖鑿的一條人工水道,便利船隻出入而得名.
    在這裡,深海的海水大量湧進內海時,帶來了極豐富的浮游生物,因而造成了各種魚群在此匯集覓食.也因為強勁的海流帶來的不只是浮游生物,還有大量的海砂,雜塵,使得這裡的水質終年不是很清澈.
    這裡的潛水是定點的,一下潛之後,潛導直接帶你到魟魚清潔站,那是指一座小小的礁石,稱做清潔站 (Cleaning station)
    巨大的蝠魟通常喜歡到此讓居住在這裡的清潔魚清除它們身上的寄生蟲.所以潛者在此都圍繞著這清潔站附近
    並且遵照規定,與礁石保持距離,(讓出現的魟魚敢靠近),耐心的等待魟魚出現.即使魟魚真的出現,也不可以迎上前去拍照或觸摸(否則會嚇走它)

    潛水客到此,最難熬的就是那枯寂無聊的等待,而且深海的海水水溫低,往往沒撐多久就已經冷的皮皮剉.能不能看到蝠魟就看運氣了

Pyramid butterflyfish (Hemitaurichthys polylepis)
正名:銀斑蝶魚
魚體中間有明顯的白色三角形圖案而易於辨別,常成群的出現在外環礁邊緣海流湍急的陡峭地形,從3米到40米深,以浮游生物為食.
因為銀斑蝶魚群聚的習性,把珊瑚礁台點綴的更美麗壯觀!!

七十島 Seventy Islands ( google map -> Seventy Islands, Palau )
洛克群島所屬的七十島(Seventy Islands)，是全帛琉從空中或水中看都最美的地方，經常被拿來當作帛琉景觀代表。在1956年被帛琉政府規劃為生態保護區，只有科學家可以進入這個區域，這裡有全帛琉最完整而豐富的海洋生態
